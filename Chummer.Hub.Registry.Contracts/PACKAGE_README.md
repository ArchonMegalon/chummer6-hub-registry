# Chummer.Hub.Registry.Contracts

Immutable contract package for Hub registry artifacts, publication and moderation workflow, release channels, install/update projections, account-aware install-linking DTOs, compatibility projections, and runtime-bundle head projections.

Consumers such as `chummer.run-services` and `chummer6-ui` depend on this package boundary for shared registry DTOs and keep orchestration, persistence, and UX-specific shaping in their own repos.
