# Chummer Run Registry Runtime

This package exposes the licensed Registry runtime services used by Chummer Hub.
Registry remains the sole owner of artifact publication, promoted release channels,
desktop release heads, install/update metadata, reviews, and compatibility truth.

The package is not a public extension API and does not grant rights beyond the
`LICENSE` file embedded in the package. Consumers must pin an immutable source
commit and verify the package digest before composition.
