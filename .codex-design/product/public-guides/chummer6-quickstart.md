# Chummer6 Quickstart Guide

Chummer6 is a campaign OS for building a runner, explaining the next rules decision, and recovering the table after the session.

## Start

1. Open the current desktop release from the downloads shelf.
2. Finish first-launch install linking only if this device should stay attached to your account.
3. Create a runner or open an existing workspace.
4. Use account and horizon rails only when the next job needs them.

## What this guide is for

- install posture
- first safe actions
- public-safe Chummer orientation

## What this guide is not

- a Shadowrun sourcebook
- copied rulebook prose
- a private campaign packet
- entitlement or payment truth
