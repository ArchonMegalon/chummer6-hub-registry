# Teable Projection Surface Spec

Teable is an operator projection mirror for Chummer state that is already safe to project.

Projection-only tables:

- `product_signals`
- `black_ledger_dispatches`
- `tick_news_delivery`
- `package_pressure`
- `ltd_adapter_readiness`

Sync direction:

`Chummer -> Teable`

Never:

`Teable -> Chummer truth`

Dry-run proof should show record shape, redacted secrets, and zero private support payload.
