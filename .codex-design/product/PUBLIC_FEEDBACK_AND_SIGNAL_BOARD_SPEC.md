# Public Feedback And Signal Board Spec

The Chummer feedback surface is a first-party signal board, not a provider-owned board.

Public jobs:

1. collect safe public feedback
2. show demand with votes/follows
3. point to roadmap when a signal becomes direction
4. point to changelog when a signal closes with proof
5. steer private issues to help/support early

Public board copy should keep one sentence explicit:

`Votes show demand. Chummer-owned proof decides what ships.`

The board must separate:

- public signal
- roadmap direction
- shipped proof
- private support
