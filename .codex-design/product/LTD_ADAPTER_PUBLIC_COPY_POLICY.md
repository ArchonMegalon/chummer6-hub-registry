# LTD Adapter Public Copy Policy

Public users should see Chummer-owned route names and Chummer-owned promises.

Allowed public phrases:

- public signal board
- vote and follow
- roadmap signal
- closeout receipt
- dispatch draft reviewed
- operator projection unavailable

Forbidden on public pages:

- ProductLift
- Syllabbles
- blipai.app
- Teable
- Emailit
- AppSumo
- LTD
- webhook
- callback secret
- env var names
- provider binding ids

Public pages may mention:

- public signal
- roadmap
- changelog
- dispatch
- package request
- account follow-up

Implementation rule:

If a provider-specific term is needed, it belongs in operator docs, not in rendered public surfaces.
